# OXR_Switcher (x32/x64)
OpenXR Runtime Switcher - GUI util for switch active OpenXR Runtime (x32/x64) with support command line. Displayed in a list everything that is installed on the system runtimes. Supported icons: Oculus, Meta, SteamVR, VirtualDesktop, WMR, Vive, Varjo, Pimax, Monado. 

OpenXR Runtime Switcher - Утилита для переключения активной среды OpenXR (x32/x64) с поддержкой командной строки. Отображает в списке все установленные в системе среды. Поддерживаются иконки: Oculus, Meta, SteamVR, VirtualDesktop, WMR, Vive, Varjo, Pimax, Monado.
